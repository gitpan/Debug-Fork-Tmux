NAME

Debug::Fork::Tmux::Config - Configuration system for Debug::Fork::Tmux

VERSION

This documentation refers to the module contained in the distribution Debug-Fork-Tmux version 1.000007.

SYNOPSIS 

    use Debug::Fork::Tmux;

    my $tmux_fqdn = Debug::Fork::Tmux->config( 'tmux_fqdn' );

DESCRIPTION

This module reads description from environment variables and use defaults if those are not set.

For example tmux_fqdn can be overridden with SPUNGE_TMUX_FQDN variable, and so on..

SUBROUTINES/METHODS

All of the following are static methods:

PUBLIC

get_config( Str the name of the option )

Retrieves configuration stored in an internal Debug::Fork::Tmux::Config constants.

Returns Str value of the configuration parameter.

PRIVATE

get_all_config_keys()

Returns Array[Str] names of all the configuration parameters.

CONFIGURATION AND ENVIRONMENT

See "CONFIGURATION AND ENVIRONMENT" in Debug::Fork::Tmux.

DIAGNOSTICS

Undefined in a configuration: <key>

Dies if no key asked was found in the configuration.
